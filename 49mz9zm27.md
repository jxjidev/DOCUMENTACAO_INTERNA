API de Pagamentos

Visao Geral

Essa API permite processar pagamentos de forma segura e eficiente.

Endpoints

POST /payments

Cria um novo pagamento.

Request Body:

{
  "amount": 100,
  "currency": "BRL"
}

Exemplo de Uso

var payment = await api.CreatePayment(new PaymentRequest
{
    Amount = 100,
    Currency = "BRL"
});
