# Exceção

Em alguns cenários, determinados clientes podem apresentar **necessidades específicas** que não se enquadram integralmente nas **regras e fluxos padrão do QADevOps**. Essas exceções normalmente estão relacionadas a **restrições técnicas**, **exigências contratuais**, **modelos de governança já estabelecidos** ou **processos internos do próprio cliente**.\
